Permit2 Widget Demo

CDN / UMD usage

Include the bundled UMD script from your deployed app (copy `dist/index.umd.js` to your CDN or app root).

<script src="https://your-app.example.com/dist/index.umd.js"></script>
<script>
  // global: Permit2Widget
  document.getElementById('cdnBtn').addEventListener('click', async () => {
    try {
      const res = await window.Permit2Widget.open('https://your-app.example.com');
      console.log('Permit result', res);
    } catch (e) { console.error(e); }
  });
</script>

npm usage (ESM)

Install with npm and import in your JS build. 

npm install @tlord101/permit2-widget

// in your code
import Permit2Widget from '@tlord101/permit2-widget';

document.getElementById('npmBtn').addEventListener('click', async () => {
  try {
    const res = await Permit2Widget.open('https://your-app.example.com');
    console.log('Permit result', res);
  } catch (e) { console.error(e); }
});
